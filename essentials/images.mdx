## Image

### Using Markdown

Use [markdown syntax](https://www.markdownguide.org/basic-syntax/#images) to add images with the following format:

```md
![title](/path/image.jpg)
```

Image files must be under 5MB. For larger files, host them on a service like [Cloudinary](https://cloudinary.com/) or [S3](https://aws.amazon.com/s3/) and embed the URL.

### Using embeds

For more control over image styling, use [embeds](/writing-content/embed):

```html
<img height="200" src="/path/image.jpg" />
```

### iFrames

Embeds another HTML page within your document. Most commonly used for videos.

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
