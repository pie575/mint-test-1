## Inline code

To denote a `word` or `phrase` as inline code, enclose it in backticks (`).


## Code blocks

Create [fenced code blocks](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks) by wrapping code in three backticks. Specify the programming language after the opening backticks for syntax highlighting, and optionally add a filename after the language identifier.

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
