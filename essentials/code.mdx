## Inline code

To mark a `word` or `phrase` as code, enclose it in backticks (`).

## Code blocks

Use [fenced code blocks](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks) by enclosing code in three backticks. Specify the programming language after the opening backticks for syntax highlighting. You can also include a filename after the language identifier.

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
