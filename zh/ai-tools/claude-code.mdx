使用 Claude Code，可在终端中编写、编辑和维护你的 Mintlify 文档。

<div id="prerequisites">
  ## 前提条件
</div>

* 已启用 Claude 订阅 (Pro、Max 或 API 访问权限)

<div id="setup">
  ## 设置
</div>

<Steps>
  <Step title="安装 Claude Code">
    ```bash
    npm install -g @anthropic-ai/claude-code
    ```
  </Step>

  <Step title="安装 Mintlify 技能">
    在你的 docs 目录中，运行：

    ```bash
    npx skills add https://mintlify.com/docs
    ```

    这会为 Claude Code 提供 Mintlify 的组件参考、写作规范
    和工作流指南。
  </Step>

  <Step title="自定义项目说明">
    编辑项目根目录中的 `AGENTS.md`，添加项目专用术语、
    风格偏好和内容边界。
  </Step>

  <Step title="开始使用 Claude Code 编写">
    ```bash
    claude
    ```
  </Step>
</Steps>
