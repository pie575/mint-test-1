使用 Cursor，在 AI 的辅助下编写、编辑和维护你的 Mintlify 文档。

<div id="prerequisites">
  ## 准备工作
</div>

* 已安装 Cursor 编辑器

<div id="setup">
  ## 设置
</div>

<Steps>
  <Step title="在 Cursor 中打开文档目录">
    打开文档仓库的根目录，也就是包含 `docs.json` 的目录。
  </Step>

  <Step title="安装 Mintlify skill">
    在集成终端中运行：

    ```bash
    npx skills add https://mintlify.com/docs
    ```

    这会为 Cursor 提供 Mintlify 的组件参考、写作规范
    和工作流指南。
  </Step>

  <Step title="自定义项目说明">
    编辑项目根目录中的 `AGENTS.md`，添加项目专用术语、
    风格偏好和内容边界。
  </Step>

  <Step title="开始使用 Cursor 编写">
    打开任意 MDX 文件，然后使用 Cursor 的 AI 功能起草和编辑文档。
  </Step>
</Steps>
