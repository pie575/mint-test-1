  ## 图片
</div>

<div id="using-markdown">
  ### 使用 Markdown
</div>

你可以使用 [Markdown 语法](https://www.markdownguide.org/basic-syntax/#images)，按以下格式添加图片：

```md
![title](/path/image.jpg)
```

请注意，图片文件大小必须小于 5MB。对于更大的文件，建议托管到 [Cloudinary](https://cloudinary.com/) 或 [S3](https://aws.amazon.com/s3/) 之类的服务上，再嵌入其 URL。

<div id="using-embeds">
  ### 使用嵌入内容
</div>

如果你想更灵活地控制图片样式，可以使用[嵌入内容](/zh/writing-content/embed)：

```html
<img height="200" src="/path/image.jpg" />
```

<div id="iframes">
  ### iFrames
</div>

在文档中嵌入另一个 HTML 页面，最常用于嵌入视频。

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
