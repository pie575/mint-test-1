<div id="text-formatting">
  ## 文本格式
</div>

支持大多数标准 Markdown 格式。你可以在文本两侧添加 `**`、`_` 或 `~` 来设置格式。

| Style | How to write it   | Result            |
| ----- | ----------------- | ----------------- |
| 粗体    | `**bold**`        | **bold**          |
| 斜体    | `_italic_`        | *italic*          |
| 删除线   | `~strikethrough~` | ~~strikethrough~~ |

这些格式也可以组合使用。例如，写成 `**_bold and italic_**` 就会得到 ***bold and italic*** 文本。

如果要设置上标和下标，请使用 HTML 标签。将文本包裹在 `<sup>` 或 `<sub>` 中。

| Text Size | How to write it          | Result                 |
| --------- | ------------------------ | ---------------------- |
| 上标        | `<sup>superscript</sup>` | <sup>superscript</sup> |
| 下标        | `<sub>subscript</sub>`   | <sub>subscript</sub>   |

<div id="linking-to-pages">
  ## 链接到页面
</div>

你可以用 `[]()` 包裹文本来添加链接。比如，写成 `[link to google](https://google.com)`，就会显示为 [link to google](https://google.com)。

链接到文档中的页面时，需要使用根相对路径。请包含完整的文件夹路径——例如，`[link to text](/writing-content/text)` 会链接到组件部分中的“Text”页面。

像 `[link to text](../text)` 这样的相对链接加载会更慢，因为它们不易被优化。

<div id="singleline">
  ### 单行
</div>

要创建块引用，请在段落前添加一个 `>`。

> 多萝西跟着她穿过了城堡里许多漂亮的房间。

```md
> 多萝西跟着她，穿过了她城堡里许多美丽的房间。
```

<div id="multiline">
  ### 多行
</div>

> 多萝西跟着她穿过城堡里许多美丽的房间。
>
> 女巫吩咐她刷洗锅壶、打扫地板，还要不断往炉火里添柴。

```md
> 多萝西跟着她穿过城堡里许多美丽的房间。
>
> 女巫吩咐她刷洗锅碗瓢盆、打扫地板，并不断往火里添柴。
```
