<div id="code-blocks">
  ## 代码块
</div>

使用[围栏代码块](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks)：用三个反引号将代码括起来，并在开头的反引号后标明代码片段所使用的编程语言，即可启用语法高亮。你也可以在编程语言后面附上代码名称。

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
