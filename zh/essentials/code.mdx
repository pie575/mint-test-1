<div id="code-blocks">
  ## 代码块
</div>

用三个反引号将代码括起来，即可创建[围栏代码块](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks)。为了启用语法高亮，请在开头的反引号后指定编程语言，并可选择在语言标识符后添加文件名。

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
