<div id="get-started-in-three-steps">
  ## 三步快速开始
</div>

在本地搭建你的文档站点，并进行首次自定义。

<div id="step-1-set-up-your-local-environment">
  ### 步骤 1：设置本地环境
</div>

<AccordionGroup>
  <Accordion icon="copy" title="在本地克隆文档仓库">
    在入门过程中，你已经为文档内容创建了一个 GitHub 仓库。你可以在 [控制台](https://dashboard.mintlify.com) 中找到它。

    如需将仓库克隆到本地，以便修改文档并预览更改，请参考 GitHub 文档中的 [Cloning a repository](https://docs.github.com/en/repositories/creating-and-managing-repositories/cloning-a-repository) 指南。
  </Accordion>

  <Accordion icon="rectangle-terminal" title="启动预览服务器">
    1. 安装 Mintlify CLI：`npm i -g mint`
    2. 进入文档目录并运行：`mint dev`
    3. 打开 `http://localhost:3000`，即可实时查看文档！

    <Tip>编辑文件时，预览会自动更新。</Tip>
  </Accordion>
</AccordionGroup>

<div id="step-2-deploy-your-changes">
  ### 步骤 2：部署你的更改
</div>

<AccordionGroup>
  <Accordion icon="github" title="安装我们的 GitHub 应用">
    前往你的 [控制台](https://dashboard.mintlify.com/settings/organization/github-app) 安装 Mintlify GitHub 应用。

    GitHub 应用会自动将你的更改部署到文档站点——无需手动部署。
  </Accordion>

  <Accordion icon="palette" title="更新你的站点名称和颜色">
    先进行一项改动：更新你的文档站点名称和颜色。

    1. 在编辑器中打开 `docs.json`。
    2. 将 `"name"` 字段改为你的项目名称。
    3. 更新 `"colors"`，使其与品牌风格保持一致。
    4. 保存后，即可立即在 `http://localhost:3000` 看到更改。

    <Tip>试着修改主色，马上就能看到明显变化！</Tip>
  </Accordion>
</AccordionGroup>

<div id="step-3-go-live">
  ### 步骤 3：正式上线
</div>

<Accordion icon="rocket" title="发布你的文档">
  1. 提交并推送更改。
  2. 你的文档很快就会更新并上线！
</Accordion>
