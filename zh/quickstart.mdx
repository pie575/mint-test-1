<div id="get-started-in-three-steps">
  ## 通过三个步骤快速上手
</div>

在本地搭建你的文档站点，并完成首次自定义。

<div id="step-1-set-up-your-local-environment">
  ### 第 1 步：设置本地环境
</div>

<AccordionGroup>
  <Accordion icon="copy" title="在本地克隆文档仓库">
    在初始引导过程中，你已经为文档内容创建了一个 GitHub 仓库。你可以在 [控制台](https://dashboard.mintlify.com) 中找到它。

    要克隆该仓库并在本地预览更改，请按照 GitHub 文档中的 [克隆仓库](https://docs.github.com/en/repositories/creating-and-managing-repositories/cloning-a-repository) 指南操作。
  </Accordion>

  <Accordion icon="rectangle-terminal" title="启动预览服务器">
    1. 安装 Mintlify 命令行界面：`npm i -g mint`
    2. 进入文档目录并运行：`mint dev`
    3. 打开 `http://localhost:3000`，即可实时查看文档！

    <Tip>编辑文件后，预览会自动更新。</Tip>
  </Accordion>
</AccordionGroup>

<div id="step-2-deploy-your-changes">
  ### 第 2 步：部署你的更改
</div>

<AccordionGroup>
  <Accordion icon="github" title="安装我们的 GitHub 应用">
    前往你的 [控制台](https://dashboard.mintlify.com/settings/organization/github-app) 安装 Mintlify GitHub 应用。

    GitHub 应用会自动部署你的更改，无需手动部署。
  </Accordion>

  <Accordion icon="palette" title="更新站点名称和配色">
    先更新文档站点的名称和配色，作为你的第一项更改。

    1. 在编辑器中打开 `docs.json`。
    2. 将 `"name"` 字段改成你的项目名称。
    3. 更新 `"colors"`，使其与你的品牌风格一致。
    4. 保存后，你就可以立即在 `http://localhost:3000` 看到更改。

    <Tip>试着修改主色，马上就能看到效果！</Tip>
  </Accordion>
</AccordionGroup>

<div id="step-3-go-live">
  ### 第 3 步：正式上线
</div>

<Accordion icon="rocket" title="发布你的文档">
  1. 提交并推送你的更改。
  2. 稍等片刻，你的文档就会更新并上线！
</Accordion>
