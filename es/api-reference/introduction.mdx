  ## Te damos la bienvenida
</div>

Puedes crear documentación de la API de dos formas: [OpenAPI](https://mintlify.com/docs/api-playground/openapi/setup) y [componentes de MDX](https://mintlify.com/docs/api-playground/mdx/configuration). Este kit de inicio utiliza la especificación de OpenAPI que aparece a continuación.

<div id="authentication">
  ## Autenticación
</div>

Todos los endpoints de la API requieren autenticación mediante tokens Bearer, tal como se define en el archivo de especificación.

```json
"security": [
  {
    "bearerAuth": []
  }
]
```
