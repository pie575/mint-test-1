Usa Cursor para escribir, editar y mantener la documentación de Mintlify con ayuda de IA.

<div id="prerequisites">
  ## Requisitos previos
</div>

* Tener instalado el editor Cursor

<div id="setup">
  ## Configuración
</div>

<Steps>
  <Step title="Abre el directorio de tu documentación en Cursor">
    Abre la raíz de tu repositorio de documentación: el directorio que contiene `docs.json`.
  </Step>

  <Step title="Instala la skill de Mintlify">
    En la terminal integrada, ejecuta:

    ```bash
    npx skills add https://mintlify.com/docs
    ```

    Esto le da a Cursor la referencia de componentes de Mintlify, sus estándares de redacción
    y orientación sobre el flujo de trabajo.
  </Step>

  <Step title="Personaliza las instrucciones del proyecto">
    Edita `AGENTS.md` en la raíz de tu proyecto para añadir terminología específica del proyecto,
    preferencias de estilo y límites de contenido.
  </Step>

  <Step title="Empieza a escribir con Cursor">
    Abre un archivo y usa las funciones de IA de Cursor para redactar y editar documentación.
  </Step>
</Steps>
