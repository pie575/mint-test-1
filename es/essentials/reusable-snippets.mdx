<div id="creating-a-custom-snippet">
  ## Crear un fragmento personalizado
</div>

**Requisito previo**: El archivo del fragmento debe estar en el directorio `snippets`.

<Note>
  Cualquier página del directorio `snippets` se tratará como un fragmento y no
  se renderizará como una página independiente. Si quieres crear una página independiente
  a partir del fragmento, importa el fragmento en otro archivo y úsalo como un
  componente.
</Note>

<div id="default-export">
  ### Exportación predeterminada
</div>

1. Agrega al archivo de fragmento el contenido que quieras reutilizar en varias
   ubicaciones. También puedes agregar variables opcionales que se rellenan mediante props
   al importarlo.

```mdx snippets/my-snippet.mdx
Hello world! This is my content I want to reuse across pages. My keyword of the
day is {word}.
```

<Warning>
  El contenido que quieras reutilizar debe estar dentro del directorio `snippets` para
  que la importación funcione.
</Warning>

2. Importa el fragmento en el archivo de destino.

```mdx destination-file.mdx
---
title: My title
description: My Description
---

import MySnippet from '/snippets/path/to/my-snippet.mdx';

## Header

Lorem ipsum dolor sit amet.

<MySnippet word="bananas" />
```

<div id="reusable-variables">
  ### Variables reutilizables
</div>

1. Exporta una variable desde tu archivo de fragmento:

```mdx snippets/path/to/custom-variables.mdx
export const myName = 'my name';

export const myObject = { fruit: 'strawberries' };
```

2. Importa la variable en el archivo de destino y úsala:

```mdx destination-file.mdx
---
title: My title
description: My Description
---

import { myName, myObject } from '/snippets/path/to/custom-variables.mdx';

Hello, my name is {myName} and I like {myObject.fruit}.
```

<div id="reusable-components">
  ### Componentes reutilizables
</div>

1. Dentro del archivo de tu fragmento, crea un componente que acepte props exportando
   una función flecha.

```mdx snippets/custom-component.mdx
export const MyComponent = ({ title }) => (
  <div>
    <h1>{title}</h1>
    <p>... snippet content ...</p>
  </div>
);
```

<Warning>
  MDX no se compila dentro del cuerpo de una función de flecha. Usa la
  sintaxis HTML siempre que puedas, o una exportación predeterminada si necesitas usar MDX.
</Warning>

2. Importa el fragmento en el archivo de destino y pásale las props:

```mdx destination-file.mdx
---
title: My title
description: My Description
---

import { MyComponent } from '/snippets/custom-component.mdx';

Lorem ipsum dolor sit amet.

<MyComponent title={'Custom title'} />
```
