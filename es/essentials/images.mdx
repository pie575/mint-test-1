  ## Imagen
</div>

<div id="using-markdown">
  ### Uso de Markdown
</div>

La [sintaxis de Markdown](https://www.markdownguide.org/basic-syntax/#images) te permite añadir imágenes con el siguiente formato:

```md
![title](/path/image.jpg)
```

Ten en cuenta que el tamaño del archivo de imagen debe ser inferior a 5 MB. Para archivos más grandes, te recomendamos alojarlos en un servicio como [Cloudinary](https://cloudinary.com/) o [S3](https://aws.amazon.com/s3/) e incrustar la URL.

<div id="using-embeds">
  ### Usar incrustaciones
</div>

Para tener más control sobre el estilo de las imágenes, puedes usar [incrustaciones](/es/writing-content/embed):

```html
<img height="200" src="/path/image.jpg" />
```

<div id="iframes">
  ### iFrames
</div>

Carga otra página HTML dentro del documento. Se usa sobre todo para incrustar videos.

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
