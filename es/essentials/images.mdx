  ## Imagen
</div>

<div id="using-markdown">
  ### Uso de Markdown
</div>

La [sintaxis de Markdown](https://www.markdownguide.org/basic-syntax/#images) te permite añadir imágenes con el siguiente formato:

```md
![title](/path/image.jpg)
```

Ten en cuenta que el tamaño del archivo de imagen debe ser inferior a 5 MB. Para archivos más grandes, te recomendamos alojarlos en un servicio como [Cloudinary](https://cloudinary.com/) o [S3](https://aws.amazon.com/s3/) e insertar la URL.

<div id="using-embeds">
  ### Uso de contenido incrustado
</div>

Para tener un mayor control sobre el estilo de las imágenes, puedes usar [contenido incrustado](/es/writing-content/embed):

```html
<img height="200" src="/path/image.jpg" />
```

<div id="iframes">
  ### iFrames
</div>

Incrusta otra página HTML dentro del documento. Se usa sobre todo para videos.

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
