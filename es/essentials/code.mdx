<div id="code-blocks">
  ## Bloques de código
</div>

Crea [bloques de código con delimitadores](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks) envolviendo el código entre tres acentos graves. Especifica el lenguaje de programación después de los acentos graves de apertura para resaltar la sintaxis y, de forma opcional, añade un nombre de archivo después del identificador del lenguaje.

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
