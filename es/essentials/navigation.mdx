El menú de navegación es la lista de enlaces que aparece en cada página de tu sitio de documentación.

Tendrás que actualizar `docs.json` cada vez que agregues una nueva página. Las páginas no se muestran automáticamente.

<div id="navigation-syntax">
  ## Sintaxis de la navegación
</div>

La sintaxis de la navegación es recursiva, lo que significa que puedes crear grupos de navegación anidados. No necesitas incluir `.mdx` en los nombres de las páginas.

<CodeGroup>
  ```json Navegación estándar
  "navigation": {
    "tabs": [
      {
        "tab": "Docs",
        "groups": [
          {
            "group": "Getting Started",
            "pages": ["quickstart"]
          }
        ]
      }
    ]
  }
  ```

  ```json Navegación anidada
  "navigation": {
    "tabs": [
      {
        "tab": "Docs",
        "groups": [
          {
            "group": "Getting Started",
            "pages": [
              "quickstart",
              {
                "group": "Nested Reference Pages",
                "pages": ["nested-reference-page"]
              }
            ]
          }
        ]
      }
    ]
  }
  ```
</CodeGroup>

<div id="folders">
  ## Carpetas
</div>

Coloca tus archivos MDX en carpetas y actualiza las rutas en `docs.json`.

Por ejemplo, para tener una página en `https://yoursite.com/your-folder/your-page`, crea una carpeta llamada `your-folder` que contenga un archivo MDX llamado `your-page.mdx`.

<Warning>
  No puedes usar `api` como nombre de una carpeta a menos que esté anidada dentro de otra. Mintlify usa Next.js, que reserva la carpeta `api` de nivel superior para llamadas internas del servidor. Se aceptaría un nombre de carpeta como `api-reference`.
</Warning>

```json Navigation With Folder
"navigation": {
  "tabs": [
    {
      "tab": "Docs",
      "groups": [
        {
          "group": "Group Name",
          "pages": ["your-folder/your-page"]
        }
      ]
    }
  ]
}
```

<div id="hidden-pages">
  ## Páginas ocultas
</div>

Los archivos MDX que no estén incluidos en `docs.json` no aparecerán en la barra lateral, pero seguirán siendo accesibles desde la barra de búsqueda y mediante enlaces directos.
