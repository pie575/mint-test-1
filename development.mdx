<Info>
  **Prerequisites**:
  - Node.js version 19 or higher
  - A docs repository with a `docs.json` file
</Info>

Follow these steps to install and run Mintlify locally on your operating system.

<Steps>
<Step title="Install the Mintlify CLI">

```bash
npm i -g mint
```
</Step>

<Step title="Preview locally">

Navigate to the directory containing your `docs.json` file and run:

```bash
mint dev
```

Your documentation will be available locally at `http://localhost:3000`.

</Step>
</Steps>

## Custom ports

By default, Mintlify uses port 3000. You can change the port with the `--port` flag. For example, to run on port 3333:

```bash
mint dev --port 3333
```

If the specified port is already in use, Mintlify automatically falls back to the next available port:

```md
Port 3000 is already in use. Trying 3001 instead.
```

## Mintlify versions

Each CLI release corresponds to a specific version of Mintlify. If your local preview doesn't match production, update the CLI:

```bash
npm mint update
```

## Validating links

The CLI can validate links in your documentation. To find broken links, run:

```bash
mint broken-links
```

## Troubleshooting

<AccordionGroup>
  <Accordion title='Error: Could not load the "sharp" module using the darwin-arm64 runtime'>

    This is usually caused by an outdated version of Node. Try the following:
    1. Remove the currently-installed version of the CLI: `npm remove -g mint`
    2. Upgrade to Node v19 or higher.
    3. Reinstall the CLI: `npm i -g mint`
  </Accordion>

  <Accordion title="Issue: Encountering an unknown error">
  
    Delete the `~/.mintlify` folder from your home directory, then run `mint dev` again.
  </Accordion>
</AccordionGroup>
