## Welcome

You can build API documentation in two ways: [OpenAPI](https://mintlify.com/docs/api-playground/openapi/setup) and [MDX components](https://mintlify.com/docs/api-playground/mdx/configuration). This starter kit uses the OpenAPI specification below.

<Card
  title="Plant Store Endpoints"
  icon="leaf"
  href="https://github.com/mintlify/starter/blob/main/api-reference/openapi.json"
>
  View the full OpenAPI specification file
</Card>

## Authentication

All API endpoints require Bearer token authentication, as shown in the specification file.

```json
"security": [
  {
    "bearerAuth": []
  }
]
```
